# VS Code 安装插件和汉化

  • 下载地址：VS Code官网链接 (opens new window)

  • 选择合适的版本，安装即可

  • 安装完成后如图

  • 安装插件

  • ctrl + shift + x，打开插件商店，输入 chinese，然后选择第一个，点击 install 其余插件 主题也是一样搜索安装

  • 安装完成后重启软件，显示如下，则汉化完成

# 通用

Chinese (Simplified) (简体中文) (opens new window) 此中文（简体）语言包为 VS Code 提供本地化界面。

翻译(英汉词典) (opens new window) 英汉词典，不依赖任何在线翻译API，无查询次数限制。可翻译驼峰和下划线命名，及对整个文件中的标识符批量翻译

# 常用必备

Bracket Pair Colorizer 2 (opens new window) 为代码中的各种结对的括号兄弟们提供了颜色高亮等功能。

any-rule (opens new window) 常用正则大全, 支持web / vscode插件2种查询方式.

# 前端方面

Live Server (opens new window) 具有实时加载功能的小型服务器

open in browser (opens new window) 可以把编辑的HTML文件等用浏览器打开，查看效果。

Preview on Web Server (opens new window) 在Web服务器上预览网页。

Quokka.js (opens new window) 在编辑器中使用JavaScript和TypeScript。

JS & CSS Minifier (Minify) (opens new window) Minify把CSS和JS压缩和削减(Minify：去掉空格回车符等)，以及把多个CSS，JS文件整合到一个文件里。不要以为你的大带宽没有必要进行这类优化。使用它的理由更重要的是文件合并，而不是压缩，而是文件整合，这样可以减少浏览器端不断发出新的连接请求，就像FTP服务器一样，多个小文件和一个大文件耗时是不一样的。

JavaScript Booster (opens new window) 代码重构工具。拥有需要代码操作，比如把var转为const或者let，去除多余的else语句，合并声明和初始化

# HTML

Auto Close Tag (opens new window) 自动添加 HTML/XML 关闭标记，与 Visual Studio IDE 或 Sublime Text 相同。

Auto Rename Tag (opens new window) 自动重命名成对的 HTML/XML 标记，与 Visual Studio IDE 相同。

HTML CSS Support (opens new window) 让 html 标签上写class 智能提示当前项目所支持的样式 新版已经支持scss文件检索

HTML Snippets (opens new window) 超级实用且初级的 H5代码片段以及提示

# CSS

CSS Peek (opens new window) 支持在源代码中的字符串中找到css/scss/less(类和id) 我的class在哪定义的，自己都找不到了，怎么办，有请这位！！

Easy LESS (opens new window) 保存时自动编译为CSS

Live Sass Compiler (opens new window) 使用实时浏览器重新加载将Sass或Scss实时编译为CSS。

Easy Sass (opens new window) 将scss转为微信小程序wxss css 支付宝小程序acss

Sass (opens new window) 缩进Sass语法突出显示、自动完成和格式化程序

px to rem & rpx (cssrem) (opens new window) 一个 px 与 rem 单位互转的 VSCode 插件，且支持WXSS微信小程序。

px2xx (opens new window) 支持px转换vw、vh、rem以及自定义转换的提示插件

ColorPicker颜色选择器 (opens new window) 颜色选择器,也支持一些常用格式

# 代码规范

EditorConfig for VS Code (opens new window) 定义代码格式的文件和一批编辑器插件，这些插件是让编辑器读取配置文件并以此来格式化代码

ESLint (opens new window) 编码规范 自动检验代码的程序，并打印检验结果：告诉你哪一个文件哪一行代码不符合哪一条编码规范，方便你去修改代码。

Prettier - Code formatter (opens new window) 使用prettier的代码格式化程序

Beautify (opens new window) 自动对html，js，css进行格式化对齐显示

驼峰翻译助手 (opens new window) 纠结怎么取变量? 中文一键翻译转换成常用大小驼峰等格式

Codelf (opens new window) 变量命名神器

Code Spell Checker (opens new window) 能够帮助我们检查单词拼写是否出现错误，检查的规则遵循 camelCase （驼峰拼写法）

Microsoft Edge Tools for VS Code (opens new window) 使用Elements for Microsoft Edge在Visual Studio代码中调试DOM

# 框架相关插件

# Vue

Vetur (opens new window) vue文件的语法高亮显示，除了支持template模板以外，还支持大多数主流的前端开发脚本和插件，比如Sass和TypeScript

Vue 3 Snippets (opens new window) 此扩展将Vue2 代码段和Vue3 代码段添加到Visual Studio代码中。

Vue Language Features (Volar) (opens new window) 与vetur相同，volar是一个针对vue的vscode插件，不过与vetur不同的是，volar提供了更为强大的功能

Vue VSCode Snippets (opens new window) 将增强Vue工作流程的代码片段

vue-helper (opens new window) 针对 Element-UI, VUX, IVIEW 的vue增强扩展

VueHelper (opens new window) Vue,Vue-router 和 Vuex 的代码片段

# React

ES7 React/Redux/GraphQL/React-Native snippets (opens new window) 一键生成react代码块以及快速补全react代码

Simple React Snippets (opens new window) React代码片段和命令的基本集合。

# 小程序支持

微信小程序开发助手 (opens new window) 微信小程序开发助手 for VSCode，代码提示 + 语法高亮

小程序助手 (opens new window) 小程序代码转换

WeApp Snippets (opens new window) 微信小程序插件 for TypeScript / wxml / config json

wechat-snippet (opens new window) 微信小程序代码辅助,代码片段自动完成

wxml (opens new window) 微信小程序wxml格式化以及高亮组件(高度自定义)

WXML - Language Services (opens new window) 微信小程序.wxml文件代码高亮，标签、属性的智能补全（同时支持原生小程序、mpvue 和 wepy 框架，并提供code snippets）

vscode weapp api (opens new window) 为 VSCode 提供微信小程序 API 提示及代码片段

vscode wxml (opens new window) 为 VSCode 提供 wxml 语法支持及代码片段

vscode-wechat (opens new window) 微信小程序支持vscode（版本v1.6.0+）

# 其他代码补全

A-super-comprehensive (opens new window) 代码补全合集(vue-webpack-bootstrap-node-mui-html-js-jquery-php-css-react-native)

JavaScript (ES6) code snippets (opens new window) ES6语法的JavaScript代码段

Echarts Enhanced Completion (opens new window) 这是一个用于编辑 echarts 的配置项时进行补全提示的 vscode 的扩展

# 可视化工具

Form Generator Plugin (opens new window) Element UI可视化表单设计及代码生成器

# Efficiency

Code Runner (opens new window) 代码一键运行，支持超过40种语言

Bookmarks (opens new window) 在代码某一行做标记，标记后该行会有书签的小图标（如图）。上下翻动的时候能够使用快捷键快速回到它的位置。

Time Master (opens new window) 自动跟踪您编码活动从而度量您编码效率的插件，它能够帮助您回顾您的编码活动，生成您的编码效率报告并提供相关建议。

carbon-now-sh (opens new window) 将代码生成漂亮图片展示

Polacode-xiaobaozi (opens new window) 基于polacode的一款vscode截图工具 📸

Image preview (opens new window) 代码中hover直接预览图片 侧边栏显示缩略图

javascript console utils (opens new window) 快速的生成console.log()，再调试js的过程中难免要打印一些东西，所以快速生成就很爽

koroFileHeader (opens new window) 用于生成文件头部注释和函数注释的插件

Turbo Console Log (opens new window) 快捷添加 console.log，一键 注释 / 启用 / 删除 所有 console.log

PicGo (opens new window) 一个快速而强大的图像上传插件，用于基于PicGo的VSCode。

vscode-faker (opens new window) 为名称，地址，lorem ipsum，商业等生成虚假数据

vscode-pdf (opens new window) 在VSCode中显示pdf文件。

# Writing

Markdown All in One (opens new window) 书写Markdown利器

Markdown Preview Enhanced (opens new window) 超级强大的 Markdown 插件。 这款插件意在让你拥有飘逸的 Markdown 写作体验。

Markdown+Math (opens new window) 在Markdown中输入数学公式(MathJax)

markdownlint (opens new window) 让你的markdown更加规范 -- Rules规则提示信息

Milkdown (opens new window) 以所见即所得的方式编辑 Markdown

# 主题

Atom One Dark Theme (opens new window) 基于Atom的一个暗黑色主题

One Dark Pro (opens new window) Atom标志性的一个黑暗主题，也是VS代码中安装最多的主题之一！

vscode-icons (opens new window) VisualStudio代码的图标
